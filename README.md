# IPEO_hurricane_assesment
Project structure: 
    - checkpoints: storage for model checkpoints
    - ipeo_hurricane_for_students: storage for datasets
    - logs: storage for training logs
    - src: 
        - models: model architectures and lightning modules
        - data_loaders: data loading utilities
        - visualizations: visualization utilities (activation layers)
        - calibration.py: calibration utilities
